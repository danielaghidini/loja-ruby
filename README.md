# Loja com Ruby on Rails

Este projeto é uma loja simples feita com Ruby on Rails no curso da Alura.
Inclui CRUD, categorias, validação de campos e localização para pt-BR.

**Instale as Gems com:**
$ bundle install

**Inicie o server com:**
$ rails s

**Versões:**
  - Ruby: 2.7.7
  - Ruby on Rails: 7.0.4
